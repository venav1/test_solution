# Тест-кейсы для создания задачи
### TC-001: Успешное создание задачи со всеми заполненными полями комбинаций Проект*Приоритет
**Предусловия:** В форме создания задачи в выпадающем списке (Исполнитель \*) доступен хотя бы один исполнитель. 
**Параметризация:** Используя параметризацию, проверить все варианты декартового произведения (Проект*Приоритет)

**Шаги:**
1. Открыть дашборд  
2. Нажать кнопку "Создать задачу"
3. Заполнить поле название: "Название_{номер_прогона}"
4. Заполнить поле описание, включая только кириллицу (прим. "Описание")
5. Выбрать "Проект"
6. Выбрать "Приоритет"
7. Выбрать "Исполнитель" 
8. Нажать кнопку "СОЗДАТЬ"

**Ожидаемый результат:** 
- На запрос https://avito-tech-internship-production.up.railway.app/api/v1/tasks/create пришел 200 код ответа
- Форма создания задачи закрылась
- На дашборде в конце списка появилась задача с наименованием "Название_{номер_прогона}"
- Под наименованием текст: "Доска: {проект} | Исполнитель: {Исполнитель}"

### TC-003: Неактивная кнопка "Создать" при незаполненном обязательном поле
**Предусловия:** В форме создания задачи в выпадающем списке (Исполнитель \*) доступен хотя бы один исполнитель. 
### **Тестовые данные (data-driven)**

| Сценарий | Пустое обязательное поле | Заполненные поля |
|--------|--------------------------|-----------------------------|
| 1      | `Название`               | `Описание`, `Проект`, `Приоритет`, `Исполнитель` |
| 2      | `Проект`                 | `Описание`, `Название`, `Приоритет`, `Исполнитель` |
| 3      | `Приоритет`              | `Описание`, `Название`, `Проект`, `Исполнитель` |
| 4      | `Исполнитель`            | `Описание`, `Название`, `Проект`, `Приоритет` |
**Шаги:**
1. Открыть дашборд  
2. Нажать кнопку "Создать задачу"
3. Оставить одно обязательное поле пустым
4. Заполнить остальные поля валидными значениями:
   - Название: "Тестовая задача" 
   - Проект: выбрать первый из списка
   - Приоритет: "Medium"
   - Исполнитель: выбрать первого из списка
   - Описание: "Описание"
5. Проверить состояние кнопки "Создать"

**Ожидаемый результат:** 
- Кнопка "Создать недоступна для клика"
