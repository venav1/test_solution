# Баг-репорты

## Баг #1: Не создается задача при пустом необязательном поле "Описание"

**Приоритет:** High

**Описание:** 
Не создается задача с заполненными обязательными полями и пустым необязательным

**Шаги воспроизведения:**
1. Открыть дашборд
2. Нажать кнопку "Создать задачу"
3. Корректно заполнить обязательные (*) поля формы
4. Не заполнять поле "Описание"
4. Нажать кнопку "Создать"

**Ожидаемый результат:** 
Задача создалась и появилась в конце списка задач на дашборде

**Фактический результат:** 
- Запрос: 'POST https://avito-tech-internship-production.up.railway.app/api/v1/tasks/create'
- Статус: 400
- Тело ответа: 
`{
    "error": "Key: 'CreateTaskRequest.Description' Error:Field validation for 'Description' failed on the 'required' tag",
    "message": "Неверный формат данных"
}`

**Окружение:**
- Браузер: Chrome

## Баг #2: Дублирование кнопки "Создать задачу" на дашборде

**Приоритет:** Medium

**Описание:** 
На странице дашборда во вкладке "Все задачи" отображаются две кнопки "Создать задачу" вместо одной.

**Шаги воспроизведения:**
1. Открыть дашборд вкладку "Все задачи"
2. Просмотреть тулбар и футбар в поисках кнопки "Создать задачу"

**Ожидаемый результат:** 
На странице должна отображаться одна кнопка "Создать задачу".

**Фактический результат:** 
На странице отображаются две кнопки "Создать задачу".

**Окружение:**
- Браузер: Chrome

---
## Баг #3: При статусе задачи "Backlog" невозможно обновить поля, доступные для редактирования 

**Приоритет:** High

**Описание:** 
При попытке обновить поля ("Название", "Описание", "Приоритет", "Исполнитель") в задаче со статусом Backlog - данные остаются прежними

**Шаги воспроизведения:**
1. Создать задачу
2. Открыть дашборд, найти эту задачу
3. Изменить все поля, доступные для редактирования, кроме статуса
4. Нажать кнопку "Обновить"

**Ожидаемый результат:** 
Данные в задаче обновились

**Фактический результат:** 
Данные в задаче не обновились: без изменения поля "Статус" (когда статус == "Backlog") не уходит PUT-запрос (https://avito-tech-internship-production.up.railway.app/api/v1/tasks/update/) на обновление полей. 

**Окружение:**
- Браузер: Chrome
---
## Баг #4: При статусах задачи "InProgress" и "Done" невозможно обновить название

**Приоритет:** High

**Описание:** 
При попытке обновить поле "Название" в задаче со статусом InProgress - название остается прежним

ЗАДАЧА1 - задача со статусом InProgress и названием (Н1)

**Шаги воспроизведения:**
1. Открыть ЗАДАЧА1 на редактирование
2. Поменять Н1 на Н2 (отличное от Н1)
3. Нажать кнопку "Обновить"

**Ожидаемый результат:** 
В поле "Название" - Н2

**Фактический результат:** 
В поле "Название" - Н1: в PUT-запросе (https://avito-tech-internship-production.up.railway.app/api/v1/tasks/update/) уходит H1.

**Окружение:**
- Браузер: Chrome

## Баг #5: Не отрабатывает фильтрация по статусу "Done"

**Приоритет:** Medium 

**Описание:** 
При фильтрации по статусу "Done" возвращаются все задачи

Предусловие: в списке задач присутствуют задачи со всеми статусами

**Шаги воспроизведения:**
1. В фильтре статус выбрать "Done"

**Ожидаемый результат:** 
Отображаются задачи только со статусом "Done"

**Фактический результат:** 
Отображаются задачи со всеми статусами

**Окружение:**
- Браузер: Chrome
